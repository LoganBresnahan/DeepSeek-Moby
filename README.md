# DeepSeek VS Code Extension

Official DeepSeek AI assistant for Visual Studio Code with perfect code formatting.

## Features

### 🤖 Intelligent Code Completion
- Context-aware inline suggestions
- Smart function and method completions
- Multi-language support

### 💬 Interactive Chat
- Sidebar chat interface
- Conversation history
- Code-aware context
- Streaming responses

### 🛠️ Powerful Code Actions
- **Explain Code** - Detailed explanations of selected code
- **Refactor Code** - Improve code structure and readability
- **Document Code** - Generate comprehensive documentation
- **Fix Bugs** - Identify and fix potential issues
- **Optimize Code** - Performance improvements
- **Generate Tests** - Create test cases

### 🎨 Perfect Formatting
- Language-specific indentation
- Code block extraction from markdown
- Integration with VS Code formatters
- Consistent style enforcement

## Installation

### Manual Installation
1. Clone this repository
2. Run `npm install`
3. Run `npm run compile`
4. Press F5 to start debugging

## Configuration

1. Get your API key from [platform.deepseek.com](https://platform.deepseek.com)
2. Open VS Code Settings (Ctrl+,)
3. Search for "DeepSeek"
4. Enter your API key in the configuration

## Usage

### Chat Interface
1. Click the DeepSeek icon in the activity bar
2. Type your question in the chat input
3. Press Enter or click Send

### Code Actions
1. Select code in the editor
2. Right-click and choose a DeepSeek action

## Privacy & Security

- Your API key is stored locally in VS Code settings
- Code and conversations stay on your machine
- No data is sent to external servers except DeepSeek API
